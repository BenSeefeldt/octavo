octavo is a minimal html+css tool for creating small 8-page booklets out of single sheets of letter size paper.

for folding instructions, search online for "one page mini book".

to save a booklet, simply save this html file with filled content. refreshing the page will clear any unsaved changes.

running your browser's print command will correctly format the pages. ensure you've set scaling to 100% and margins to 0.

this file is standalone, and can be saved and distributed with no additional dependencies.
